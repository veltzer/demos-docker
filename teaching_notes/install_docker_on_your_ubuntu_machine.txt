	install docker.io:
		$ sudo apt install docker.io
	add yourself to the docker group:
		$ sudo adduser [yourusername] docker
	restart your VM for the changes to take effect
	login
	check that you are a member of the docker group:
		$ groups
	and see that "docker" is in the output
	see that commands like:
		$ docker images
		$ docker container ps
	work...
