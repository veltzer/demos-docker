Create a docker image that prints "Hello, world!" and exists
Try to use a language other than python if you know one...

Reminders...
- You have to write a file called "Dockerfile" with instruction.
- In the dockerfile you have to state on which image your image is built upon.
- Use docker hub (internet) to find the best image to base your image on.
- Write you app.
- Add your app to the image in the dockerfile.
- Tell docker using the CMD instruction how to run your app.
- Build your app using "docker build"
- Run your app using "docker run"
- Kill you app using "docker kill"
